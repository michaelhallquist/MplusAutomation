# Extract output of R3STEP procedure

Extract output of R3STEP procedure

## Usage

``` r
extractR3step(outfiletext, filename)
```

## Arguments

- outfiletext:

  character vector of Mplus output file from which to extract the AUX
  section

- filename:

  filename of the Mplus output file being processed

## Value

A list containing the parsed R3STEP sections
