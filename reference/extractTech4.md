# Extract Technical 4 matrix from Mplus

Function that extracts the Tech4 matrix

## Usage

``` r
extractTech4(outfiletext, filename)
```

## Arguments

- outfiletext:

  the text of the output file

- filename:

  The name of the file

## Value

A list of class “mplus.tech4”
