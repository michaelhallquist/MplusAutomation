# Small helper function to obtain number of factors for an EFA output section

Small helper function to obtain number of factors for an EFA output
section

## Usage

``` r
get_efa_nfactors(headers)
```

## Arguments

- headers:

  a vector of EFA section headers from which to parse number of factors

## Value

A vector of the number of factors in each heading, or a matrix if
multilevel output
