# Extract Technical 9 matrix from Mplus

Function that extracts the Tech9 matrix

## Usage

``` r
extractTech9(outfiletext, filename)
```

## Arguments

- outfiletext:

  the text of the output file

- filename:

  The name of the file

## Value

A list of class “mplus.tech9”
