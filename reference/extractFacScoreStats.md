# Extract Factor Score Statistics

Function for extracting matrices for factor scores

## Usage

``` r
extractFacScoreStats(outfiletext, filename)
```

## Arguments

- outfiletext:

  The text of the output file

- filename:

  The name of the output file

## Value

A list
