# Extract Technical 1 matrix from Mplus

Function that extracts the Tech1 matrix

## Usage

``` r
extractTech1(outfiletext, filename)
```

## Arguments

- outfiletext:

  the text of the output file

- filename:

  The name of the file

## Value

A list of class “mplus.tech1”
