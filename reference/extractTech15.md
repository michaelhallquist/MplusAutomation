# Extract Technical 15 from Mplus

The TECH15 option is used in conjunction with TYPE=MIXTURE to request
conditional probabilities for the latent class variables.

## Usage

``` r
extractTech15(outfiletext, filename)
```

## Arguments

- outfiletext:

  the text of the output file

- filename:

  The name of the file

## Value

A list of class “mplus.tech15”
