# Extract the summaries from one file

Description: This function parses an output file for specific model
details. It returns a list of model details for a single output file.

## Usage

``` r
extractSummaries_1file(outfiletext, filename, input)
```

## Arguments

- outfiletext:

  This is the output file in string form to be parsed. Passed in from
  extractModelSummaries.

- filename:

  Name of the file being parsed. Used in case of bad model, prints a
  warning.

## Value

A list of the summaries
