# Reconstruct matrix from unlabeled multi-line text output

worker function for extracting Mplus matrix output from an unlabeled
section where matrices are spread across blocks to keep within width
constraints example: class counts output from latent class models.

## Usage

``` r
unlabeledMatrixExtract(outfiletext, filename)
```

## Arguments

- outfiletext:

  The text of the output file

- filename:

  The name of the output file

## Value

a matrix
