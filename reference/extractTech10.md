# Extract Technical 10 matrix from Mplus

Function that extracts the Tech10 matrix

## Usage

``` r
extractTech10(outfiletext, filename)
```

## Arguments

- outfiletext:

  the text of the output file

- filename:

  The name of the file

## Value

An empty list
