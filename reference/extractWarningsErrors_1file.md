# Extract warnings and errors from 1 mplus file

Helper function

## Usage

``` r
extractWarningsErrors_1file(outfiletext, filename, input)
```

## Arguments

- outfiletext:

  The text of the output file

- filename:

  The filename

- input:

  The input

## Value

A list with two elements

- `errors`: Mplus Errors

- `warnings`: Mplus Warnings
