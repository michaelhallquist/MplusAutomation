# Extract free file output

Function for reading "free" output where a sequence of values populates
a matrix

## Usage

``` r
extractFreeFile(filename, outfile, make_symmetric = TRUE)
```

## Arguments

- filename:

  The name of the output file

- outfile:

  The output file

- make_symmetric:

  A logical indicating whether or not to make the matrix symmetric,
  defaults to `TRUE`

## Value

a matrix
